Accountability: About This Data Set

Brief: This dataset contains 41396 Airline Reviews, which covered almost every airline company all over the world, with 20 variables in total.

In these variables, 11 variables is in String type. They are "airline_name", "link", "title", "author", "author_country", "date", "content", "aircraft", "type_traveller","cabin_flown" and "route".

And, the other variables are numeric. They are "overall_rating", "seat_comfort_rating", "cabin_staff_rating", "food_beverages_rating", "infilght_entertainment_rating", "ground_service_rating", "wifi_connectivity_rating", "value_money_rating" and "recommended".

Where did this data come from?

I found this data from a github page, Skytrax User Reviews Dataset (https://github.com/quankiquanki/skytrax-reviews-dataset), via redirecting from the Awesome Public Datasets page (https://github.com/awesomedata/awesome-public-datasets).

As the author described in the github page, this dataset was created from all user reviews found on Skytrax (www.airlinequality.com).

Who has access to it?

As the author described in the github page, These reviews are accessible by anyone with a browser and the robots.txt on their website did not specifically prohibit the scraping of them.
